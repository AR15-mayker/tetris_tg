### 🧑‍💻 Участник 1: Инициализация и основной цикл
Создание файла main.py, инициализация pygame
Настройка окна, FPS, обновления экрана
Управление состояниями игры: меню, игра, пауза, проигрыш
Объединение всех частей в единый запускаемый файл

Реализация класса Tetromino (tetromino.py)
Генерация случайных фигур
Логика поворотов (rotate), движения влево/вправо/вниз
Логика проверки коллизий (стены, пол, другие блоки)
Проверка завершения игры (если фигура достигла верхней границы)

Файл renderer.py: отрисовка игрового поля, текущей фигуры
Отображение следующей фигуры, счёта, уровня
Отрисовка текста (шрифт, очки, Game Over)
Визуальная сетка, цвета, оформление окна
Анимации при удалении строк (опционально)

### 🔍 Участник 4: Управление и тестирование
Файл input_handler.py: обработка клавиш

Пробел – ускоренное падение
Esc – пауза
Подключение управления к игровому циклу
Тестирование: проверка багов, переполнений, выхода за границы
Поиск дубликатов кода, предложение рефакторинга
